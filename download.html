Inštalácia

Inštalácia za 3 kroky

Od stiahnutia po bežiacu aplikáciu v system tray — za menej než minútu.

1
Stiahnite inštalátor Kliknite na tlačidlo vyššie a uložte súbor iBlockySync-Setup.exe. Po dokončení sťahovania ho otvorte.
2
Spustite a potvrďte SmartScreen Ak sa zobrazí modré okno Windows SmartScreen, kliknite na „Ďalšie informácie" → „Spustiť aj tak". Inštalátor je podpísaný, takže ide o bezpečné a očakávané správanie. .NET 8.0 je súčasťou — nič ďalšie inštalovať netreba.
3
Aplikácia beží v system tray Po inštalácii sa iBlocky Windows Companion spustí na pozadí a jeho ikona sa objaví v systémovej lište (vpravo dole pri hodinách). Žiadne okno sa neotvorí — aplikácia je pripravená synchronizovať.
Posledný krok — prepojte s iPhone

Aplikácia ešte potrebuje váš API kľúč z iBločky, aby vedela, kam doklady patria. Skopírujte ho z iBločky → Nastavenia → Windows Sync a vložte do iBlocky Windows Companion. Celý postup vrátane prihlásenia do Pohody nájdete v sprievodcovi nastavením.

Installation

Install in 3 steps

From download to a running system-tray app — in under a minute.

1
Download the installer Click the button above and save iBlockySync-Setup.exe. Once the download finishes, open it.
2
Run it and confirm SmartScreen If the blue Windows SmartScreen dialog appears, click "More info" → "Run anyway". The installer is signed, so this is safe and expected. .NET 8.0 is included — nothing else to install.
3
The app runs in the system tray After installation iBlocky Windows Companion starts in the background and its icon appears in the system tray (bottom-right near the clock). No window opens — the app is ready to sync.
Last step — connect with your iPhone

The app still needs your API key from iBločky so it knows where documents belong. Copy it from iBločky → Settings → Windows Sync and paste it into iBlocky Windows Companion. The full procedure, including the Pohoda login, is in the setup guide.
