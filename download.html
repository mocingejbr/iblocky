Nastavenie za menej než 10 minút

Päť krokov od stiahnutia po automatický import dokladov do Pohody.

1
Stiahnite inštalátor Kliknite na tlačidlo vyššie a uložte súbor iBlockyWindowsCompanion-Setup.exe. Po dokončení sťahovania ho otvorte.
2
Nainštalujte a potvrďte SmartScreen Ak sa zobrazí modré okno Windows SmartScreen, kliknite na „Ďalšie informácie" → „Spustiť aj tak". Inštalátor je podpísaný — ide o bezpečné a očakávané správanie. .NET 8.0 je súčasťou inštalátora. Po inštalácii sa automaticky zobrazí sprievodca nastavením (kroky 3 a 4). Pri bežnom spustení beží aplikácia na pozadí — ikona v systémovej lište (vpravo dole pri hodinách), bez vlastného okna.
3
API kľúč Sprievodca nastavením vás vyzve na zadanie API kľúča. Otvorte iPhone aplikáciu iBločky, prejdite do Nastavenia → Pohoda Sync a skopírujte zobrazený API kľúč. Vložte ho do sprievodcu a potvrďte.
4
Prihlasovacie údaje Pohody V tom istom sprievodcovi zadáte meno a heslo k vašej Pohode. API kľúč aj prihlásenie do Pohody môžete neskôr zmeniť v nastaveniach Windows Companion. IČO zadávať netreba — Windows Companion ho načítava automaticky z Pohody. Dôležité: každý priečinok v iBločky musí mať v názve správne IČO a zodpovedajúca firma musí byť vytvorená v Pohode — inak sa doklady z neho neimportujú.
5
Overenie pripojenia Späť v iPhone aplikácii klepnite na „Skontrolovať pripojenie". Pri úspešnom prepojení sa zobrazí stav „Pripojené". Hotovo — import dokladov prebieha automaticky každých 30 sekúnd.
Riešenie problémov

Časté otázky k inštalácii

Windows zobrazil varovanie SmartScreen — je to bezpečné?

Áno. Inštalátor je digitálne podpísaný cez SignPath Foundation. SmartScreen zobrazuje upozornenie pri každej novšej aplikácii, kým si nevybuduje reputáciu u Microsoftu. Kliknite na „Ďalšie informácie“ a potom „Spustiť aj tak“.

Potrebujem nainštalovať .NET?

Nie. .NET 8.0 je zabalený priamo v inštalátore. Nemusíte inštalovať žiadne ďalšie závislosti — stačí spustiť iBlockyWindowsCompanion-Setup.exe.

Ako zistím, že aplikácia beží?

iBločky Windows Companion beží na pozadí bez okna. Jeho ikona sa zobrazí v system tray (vpravo dole). Po vložení API kľúča uvidíte v aplikácii iBločky na iPhone živý stav „Windows pripojený“.

Kde nájde aplikácia Pohoda.exe?

iBločky Windows Companion automaticky deteguje cestu k Pohoda.exe pri prvom spustení. Ak detekcia zlyhá, v nastaveniach kliknite na Browse a vyberte súbor ručne — štandardne v C:\Program Files\STORMWARE\POHODA\.

Funguje to s viacerými firmami?

Áno. iBločky Windows Companion automaticky mapuje IČO z každého bločku na správnu databázu Pohody. Každý priečinok v iBločky musí mať v názve správne IČO firmy, aby mapovanie fungovalo.

Digitálne podpisovanie kódu

Digitálny podpis tohto softvéru je zabezpečený prostredníctvom SignPath Foundation — neziskovej organizácie, ktorá poskytuje bezplatnú infraštruktúru na podpisovanie kódu pre open-source projekty. Podpísaný kód zaisťuje, že inštalátor nebol po zverejnení pozmenený.

Code signing for this software is provided by SignPath Foundation, a non-profit organization providing free code signing infrastructure for open-source projects.

Set up in under 10 minutes

Five steps from download to automatic document imports into Pohoda.

1
Download the installer Click the button above and save iBlockyWindowsCompanion-Setup.exe. Once the download finishes, open it.
2
Install and confirm SmartScreen If the blue Windows SmartScreen dialog appears, click "More info" → "Run anyway". The installer is signed — this is safe and expected. .NET 8.0 is bundled in the installer. After installation a setup wizard launches automatically (steps 3 and 4). On normal startup the app runs in the background — icon in the system tray (bottom-right near the clock), no dedicated window.
3
API key The setup wizard prompts you for an API key. Open the iBločky iPhone app, go to Settings → Pohoda Sync, and copy the displayed API key. Paste it into the wizard and confirm.
4
Pohoda credentials In the same wizard you enter your Pohoda username and password. Both the API key and Pohoda credentials can be changed later in Windows Companion settings. You don't need to enter any IČO — Windows Companion fetches it automatically from Pohoda. Important: each folder in iBločky must have the correct IČO in its name, and that company must exist in Pohoda — otherwise its documents won't be imported.
5
Verify connection Back in the iPhone app tap "Check connection". On success you'll see the status "Connected". Done — document imports run automatically every 30 seconds.
Troubleshooting

Installation FAQ

Windows showed a SmartScreen warning — is it safe?

Yes. The installer is digitally signed via SignPath Foundation. SmartScreen warns about every newer app until it builds reputation with Microsoft. Click “More info” and then “Run anyway”.

Do I need to install .NET?

No. .NET 8.0 is bundled right in the installer. You don't need any other dependencies — just run iBlockyWindowsCompanion-Setup.exe.

How do I know the app is running?

iBločky Windows Companion runs in the background with no window. Its icon shows in the system tray (bottom-right). Once you've entered the API key, the iBločky app on your iPhone shows a live “Windows connected” status.

Where does the app find Pohoda.exe?

iBločky Windows Companion automatically detects the path to Pohoda.exe on first launch. If detection fails, click Browse in settings and pick the file manually — typically in C:\Program Files\STORMWARE\POHODA\.

Does it work with multiple companies?

Yes. iBločky Windows Companion automatically maps the IČO from each receipt to the right Pohoda database. Every folder in iBločky must have the correct company IČO in its name for the mapping to work.
