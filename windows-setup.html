Sprievodca nastavením
Windows Sync

Prepojte iBločky s účtovným softvérom Pohoda v niekoľkých krokoch. Celé nastavenie vám zaberie menej ako 10 minút.

Toto je cesta pre automatický import (Pro Max). Inštalátor iBlockySync nájdete na stránke na stiahnutie.

1
Predplatné
Aktivujte predplatné Pro Max

Windows Sync je dostupný výhradne v rámci predplatného Pro Max. Aktivujte ho priamo v aplikácii iBločky v sekcii Nastavenia → Predplatné.

2
Inštalácia
Stiahnite aplikáciu pre Windows

Stiahnite si inštalátor iBlockySync z odkazu v aplikácii alebo z našej webovej stránky. Aplikácia je dostupná ako 64-bitový balíček pre Windows 10 a novší.

Windows 10 / 11 .NET 8.0 (zahrnuté) 64-bit
3
Inštalácia
Nainštalujte aplikáciu

Spustite stiahnutý inštalátor a dokončite inštaláciu podľa pokynov sprievodcu. Po inštalácii spustite iBlockySync — ikona sa zobrazí v systémovej lište.

4
Konfigurácia
Skopírujte API kľúč

V mobilnej aplikácii iBločky prejdite do Nastavenia → Windows Sync a skopírujte zobrazený API kľúč. Tento kľúč slúži na bezpečné overenie identity vášho zariadenia.

5
Konfigurácia
Zadajte API kľúč do Windows aplikácie

V aplikácii iBlockySync otvorte Nastavenia a vložte skopírovaný API kľúč do príslušného poľa. Kľúč uložte tlačidlom Potvrdiť.

6
Konfigurácia
Zadajte prihlasovacie údaje Pohody

Zadajte používateľské meno a heslo pre vašu inštaláciu Pohody. Odporúčame použiť administrátorský účet alebo účet s prístupom ku všetkým firmám, s ktorými plánujete pracovať.

7
Overenie
Otestujte pripojenie

V aplikácii iBločky klepnite na Otestovať pripojenie. Aplikácia overí komunikáciu so Windows aplikáciou iBlockySync aj prihlasovacie údaje do Pohody. Po úspešnom teste sa zobrazí stav "Windows pripojený".

✓
Export
Exportujte prvý doklad

Nastavenie je dokončené. Otvorte ľubovoľný doklad v iBločky a použite funkciu Exportovať do Pohody. Doklad sa automaticky odošle do vašej inštalácie Pohody na počítači.

⚠
Dôležité
Správne IČO firmy je povinné

Doklady do Pohody je možné exportovať iba z priečinka firmy, ktorá má v iBločky správne zadané IČO. Pohoda používa IČO na priradenie dokladu ku konkrétnej účtovnej jednotke. Pred prvým exportom skontrolujte nastavenia firmy v iBločky (Firma → Upraviť → IČO).

Všetko nastavené

Pripravení na export

Ak ste úspešne dokončili všetky kroky, môžete sa vrátiť do aplikácie a začať exportovať doklady do Pohody.
