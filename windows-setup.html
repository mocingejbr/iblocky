Windows Sync

Prepojte iBločky s účtovným softvérom Pohoda v niekoľkých krokoch. Celé nastavenie vám zaberie menej ako 10 minút.

Toto je cesta pre automatický import (Pro Max). Inštalátor iBločky Windows Companion nájdete na stránke na stiahnutie.

1
Predplatné
Aktivujte predplatné Pro Max

Windows Sync je dostupný výhradne v rámci predplatného Pro Max. Aktivujte ho priamo v aplikácii iBločky v sekcii Nastavenia → Predplatné.

2
Inštalácia
Stiahnite aplikáciu pre Windows

Stiahnite si inštalátor iBločky Windows Companion z odkazu v aplikácii alebo z našej webovej stránky. Aplikácia je dostupná ako 64-bitový balíček pre Windows 10 a novší.

Windows 10 / 11 .NET 8.0 (zahrnuté) 64-bit
3
Inštalácia
Nainštalujte aplikáciu

Spustite stiahnutý inštalátor a dokončite inštaláciu podľa pokynov sprievodcu. Po inštalácii spustite iBločky Windows Companion — ikona sa zobrazí v systémovej lište.

4
Konfigurácia
Skopírujte API kľúč

V mobilnej aplikácii iBločky prejdite do Nastavenia → Windows Sync a skopírujte zobrazený API kľúč. Tento kľúč slúži na bezpečné overenie identity vášho zariadenia.

5
Konfigurácia
Zadajte API kľúč do Windows aplikácie

V aplikácii iBločky Windows Companion otvorte Nastavenia a vložte skopírovaný API kľúč do príslušného poľa. Kľúč uložte tlačidlom Potvrdiť.

6
Konfigurácia
Zadajte prihlasovacie údaje Pohody

Zadajte používateľské meno a heslo pre vašu inštaláciu Pohody. Odporúčame použiť administrátorský účet alebo účet s prístupom ku všetkým firmám, s ktorými plánujete pracovať.

7
Overenie
Otestujte pripojenie

V aplikácii iBločky klepnite na Otestovať pripojenie. Aplikácia overí komunikáciu s iBločky Windows Companion aj prihlasovacie údaje do Pohody. Po úspešnom teste sa zobrazí stav "Windows pripojený".

✓
Export
Exportujte prvý doklad

Nastavenie je dokončené. Otvorte ľubovoľný doklad v iBločky a použite funkciu Exportovať do Pohody. Doklad sa automaticky odošle do vašej inštalácie Pohody na počítači.

⚠
Dôležité
Správne IČO firmy je povinné

Doklady do Pohody je možné exportovať iba z priečinka firmy, ktorá má v iBločky správne zadané IČO. Pohoda používa IČO na priradenie dokladu ku konkrétnej účtovnej jednotke. Pred prvým exportom skontrolujte nastavenia firmy v iBločky (Firma → Upraviť → IČO).

Všetko nastavené

Pripravení na export

Ak ste úspešne dokončili všetky kroky, môžete sa vrátiť do aplikácie a začať exportovať doklady do Pohody.

Windows Sync
setup guide

Connect iBločky with the Pohoda accounting software in a few steps. The whole setup takes less than 10 minutes.

This is the path for automatic import (Pro Max). You'll find the iBločky Windows Companion installer on the download page.

1
Subscription
Activate the Pro Max subscription

Windows Sync is available exclusively with the Pro Max subscription. Activate it directly in the iBločky app under Settings → Subscription.

2
Installation
Download the Windows app

Download the iBločky Windows Companion installer from the link in the app or from our website. The app ships as a 64-bit package for Windows 10 and later.

Windows 10 / 11 .NET 8.0 (included) 64-bit
3
Installation
Install the app

Run the downloaded installer and complete the installation following the wizard. After installing, launch iBločky Windows Companion — its icon appears in the system tray.

4
Configuration
Copy the API key

In the iBločky mobile app go to Settings → Windows Sync and copy the displayed API key. This key securely verifies your device's identity.

5
Configuration
Enter the API key in the Windows app

In iBločky Windows Companion open Settings and paste the copied API key into the field. Save it with the Confirm button.

6
Configuration
Enter your Pohoda credentials

Enter the username and password for your Pohoda installation. We recommend an administrator account, or an account with access to every company you plan to work with.

7
Verification
Test the connection

In the iBločky app tap Test connection. It verifies communication with iBločky Windows Companion and your Pohoda credentials. On success you'll see the "Windows connected" status.

✓
Export
Export your first document

Setup is complete. Open any document in iBločky and use Export to Pohoda. The document is sent automatically to your Pohoda installation on the PC.

⚠
Important
A correct company ID (IČO) is required

Documents can only be exported to Pohoda from a company folder that has the correct IČO set in iBločky. Pohoda uses the IČO to match a document to the right accounting entity. Before your first export, check the company settings in iBločky (Company → Edit → IČO).

All set

Ready to export

If you've completed all the steps, head back to the app and start exporting documents to Pohoda.
